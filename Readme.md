# 📧 Spam SMS Detector

A Machine Learning web application that detects whether an SMS message is Spam or Ham.

## Features

- Spam Detection
- Confidence Score
- Probability Visualization
- TF-IDF Vectorization
- Multinomial Naive Bayes Model
- Streamlit Web Interface

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- NLTK
- Joblib

## Dataset

UCI SMS Spam Collection

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
